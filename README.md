# Rate My Protein

## პროექტის სათაური

**პროტეინის საკვები დანამატების შეფასებისა და გემოს ანალიზის ვებ-პლატფორმა**

## Project Title

**Protein Supplement Taste Analysis and Review Platform**

---

## პროექტის აღწერა

Rate My Protein არის ვებ-პლატფორმა, რომელიც მომხმარებლებს საშუალებას აძლევს შეაფასონ პროტეინის საკვები დანამატები არა მხოლოდ ზოგადი რეიტინგით, არამედ გემოსთან დაკავშირებული დეტალური მახასიათებლებით.

პლატფორმის მიზანია დაეხმაროს მომხმარებლებს პროტეინის დანამატის არჩევაში რეალური მომხმარებლების გამოცდილების საფუძველზე. ხშირად პროდუქტის ყიდვამდე მომხმარებელს არ აქვს საკმარისი ინფორმაცია იმაზე, რამდენად სასიამოვნო გემო აქვს პროტეინს, რამდენად კარგად იხსნება, ზედმეტად ტკბილია თუ არა, აქვს თუ არა ხელოვნური aftertaste და რასთან ერთად ჯობია მისი მიღება — წყალთან, რძესთან თუ სხვა სასმელთან.

სისტემა მომხმარებლებს მისცემს შესაძლებლობას დაათვალიერონ სხვადასხვა ბრენდის პროტეინის პროდუქტები, წაიკითხონ შეფასებები, დატოვონ საკუთარი მიმოხილვა და შეაფასონ პროდუქტი სტრუქტურირებული კრიტერიუმებით.

---

## პრობლემა

პროტეინის საკვები დანამატები ფართოდ გამოიყენება სპორტით, ფიტნესითა და ჯანსაღი კვებით დაინტერესებული ადამიანების მიერ. მიუხედავად ამისა, პროდუქტის არჩევა ხშირად რთულია, რადგან ბევრი პლატფორმა ძირითადად ყურადღებას ამახვილებს ზოგად რეიტინგზე ან კვებით შემადგენლობაზე.

გემო, ტექსტურა, ხსნადობა, სიტკბო და aftertaste ხშირად არ არის შეფასებული სტრუქტურირებულად. შედეგად, მომხმარებელი შეიძლება შეიძინოს პროდუქტი კარგი შემადგენლობით, მაგრამ არასასიამოვნო გემოთი ან ცუდი ხსნადობით.

- სიტკბოს დონე
